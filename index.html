What is Hypatia Engine?

Hypatia Engine is a game engine, currently a framework/library in alpha, for 2D tile-based adventure games. It currently has the following features:

  • Easy management of stateful sprites, e.g., a character's sprites for walking around
  • Live updating of sprites/animations using the anchor system
  • Tile map system; save, load, modify
